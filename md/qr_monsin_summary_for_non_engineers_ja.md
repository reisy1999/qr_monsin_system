# ご担当者様へ：QR問診システムの概要と安全性について

この資料では、新しく導入する「QR問診システム」がどのようなもので、なぜ安全に使えるのかを、専門用語を避けて分かりやすくご説明します。

---

## 1. このシステムで何ができる？

このシステムは、患者様がご自身のスマートフォンで問診票を入力し、その結果をQRコードに変換して、受付で提示していただくためのものです。

**目的:**
- 患者様の待ち時間を短縮し、利便性を向上させる。
- 問診票のペーパーレス化を進め、管理コストを削減する。
- 高いセキュリティを確保し、患者様の情報を安全に取り扱う。

**登場するアプリは2つだけです:**
1.  **患者様用アプリ (`input-app`):** スマートフォンで問診票に回答し、QRコードを生成します。
2.  **院内用アプリ (`restore-app`):** 受付のパソコンでQRコードを読み取り、内容をテキストファイルで表示・保存します。

---

## 2. 簡単な使い方

**【患者様の操作】**
1. 院内やホームページで案内されたURLに、ご自身のスマートフォンでアクセスします。
2. 画面の指示に従って、問診の質問に回答します。
3. 全て回答し終えると、画面にQRコードが表示されます。
4. 受付でそのQRコードを提示します。

**【受付スタッフ様の操作】**
1. 院内専用のパソコンで「QR問診読み取りアプリ」を起動します。
2. 患者様が提示したQRコードを、専用のバーコードリーダーで読み取ります。
3. 自動的に問診内容が記載されたテキストファイル（`questionnaire.txt`）がダウンロードされます。
4. そのファイルを開き、内容を確認したり、印刷したりします。

---

## 3. なぜこのシステムは安全なのですか？

このシステムの最大の特徴は、**患者様の情報をインターネットに一切流さない**という点にあります。

| 安全性のポイント | 説明 |
| :--- | :--- |
| **情報はスマホの外に出ない** | 問診の回答は、全て患者様のスマホの中で処理され、QRコードに変換されます。**インターネットやサーバーに情報が送信されることは一切ありません。** |
| **QRコードは「鍵付きの金庫」** | 生成されるQRコードは、中身が分からないように**高度に暗号化**されています。これは、まるで鍵のかかった金庫のようなものです。万が一、QRコードが第三者に見られても、中身を解読することはできません。 |
| **鍵は病院のPCにしかない** | 暗号化されたQRコード（金庫）を開けるための**「鍵」は、院内の専用PCにしかありません。**この鍵は外部のネットワークと繋がっていないため、盗まれる心配もありません。 |
| **データは表示後すぐに消去** | 受付で問診内容をテキストファイルとして保存した後、**アプリ上のデータは即座に消去されます。** パソコンのメモリにも一切情報が残らないため、非常に安全です。 |
| **なりすまし・改ざん防止** | QRコードには「本物であることの印」が付けられています。もし誰かが偽のQRコードを作ったり、内容を書き換えたりしても、アプリが瞬時に見破り、受け付けない仕組みになっています。 |

### 特に重要なこと

このシステムは、患者様のプライバシーを最高レベルで守るために、あえて「不便」な設計にしている部分があります。

- **院内PCはインターネットから隔離:** 安全性を高めるため、読み取り用のPCは外部のネットワークから切り離されています。
- **データを一切保存しない:** アプリは、一度使ったら情報を忘れる「使い捨て」の設計です。これにより、情報漏洩のリスクを根本からなくしています。

---

## まとめ

QR問診システムは、患者様の利便性を高めつつ、**「情報を外に出さない」「見られても解読できない」「使ったらすぐ消す」**という3つの原則を徹底することで、紙の問診票よりも安全な情報管理を実現しています。
